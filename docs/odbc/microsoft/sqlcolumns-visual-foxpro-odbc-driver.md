# SQLColumns (Visual FoxPro ODBC Driver)
> [!NOTE]  
>  This topic contains Visual FoxPro ODBC Driver-specific information. For general information about this function, see the appropriate topic under [ODBC API Reference](../../odbc/reference/syntax/odbc-api-reference.md).  
  
 Support: Full  
  
 ODBC API Conformance: Level 1  
  
 For a table, creates a result set that is the column list for the specified table or tables.  
  
 For more information, see [SQLColumns](../../odbc/reference/syntax/sqlcolumns-function.md) in the *ODBC Programmer's Reference*.
